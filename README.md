# School Application Form - Tkinter GUI

A student enrollment management system built with Python's Tkinter library. Collects applicant information (personal details, parental information, fees, contact) and stores submissions in a text file with consent verification.

##  Features

- Student Information Form - Captures full name, age, contact details
- Parental Data - Mother's and father's names, occupations
- Enrollment Details - Fees amount, address, contact number
- Consent Verification - Checkbox ensures guardian approval before submission
- File Storage - Automatically saves applications to `school_application.txt.`
- Form Reset - Clears all fields after successful submission
- User Feedback - Console output confirms submission status
